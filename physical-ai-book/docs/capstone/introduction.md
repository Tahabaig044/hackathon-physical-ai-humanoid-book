# Capstone: Autonomous Humanoid

This capstone module brings together all concepts learned in previous modules to design and implement an autonomous humanoid robot system.

## Project Overview

The autonomous humanoid project integrates ROS 2, simulation environments, NVIDIA Isaac, and VLA models to create a capable humanoid robot.

## Learning Objectives

- Integrate multiple robotics frameworks
- Implement perception-action systems
- Deploy on physical or simulated hardware
