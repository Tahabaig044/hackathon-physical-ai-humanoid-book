# Introduction to Gazebo & Unity

This module covers simulation environments for robotics using Gazebo and Unity. Simulation is crucial for testing robotics algorithms before deployment on real hardware.

## Gazebo

Gazebo provides realistic physics simulation, high-quality graphics, and programmatic interfaces. It's widely used in the robotics community.

## Unity

Unity is a powerful game engine that can be used for robotics simulation with the Unity Robotics Hub.
